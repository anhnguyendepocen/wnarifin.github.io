Sample Size Calculator (web)

Under development.

The Sample Size Calculator (web) consists of:

  1. Means.
    • Single mean.
    • Two-mean comparison (independent)
    • Two-mean comparison (paired) and standard deviation of difference.
  2. Proportions.
    • Single proportion.
    • Two-proportion comparison (independent).
    • Two-proportion comparison (paired) [McNemar's test].
    • Sensitivity and specificity.
  3. Pearson's correlation.
  4. Intraclass correlation coefficient (ICC).
  5. Kappa coefficient.
  6. Cronbach's alpha coefficient.

I am in the process of converting most of the available tabs/calculators in the spreadsheet version into the web version with Javascripts. The web version will have more practical features than the spreadsheet version (which is used mostly for teaching purposes).
