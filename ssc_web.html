Sample Size Calculator (web)

Under development.

The Sample Size Calculator (web) consists of:

  1. Means.
    • Single mean.
    • Two-mean comparison (independent).
    • Two-mean comparison (paired) and standard deviation of difference.
  2. Proportions.
    • Single proportion.
    • Two-proportion comparison (independent).
    • Two-proportion comparison (paired) [McNemar's test].
    • Sensitivity and specificity.
  3. Pearson's correlation.
  4. Intraclass correlation coefficient (ICC).
  5. Kappa coefficient.
  6. Cronbach's alpha coefficient.
  7. Sample size for animal study

I am in the process of converting most of the available tabs/calculators in the spreadsheet version into this web version using JavaScript. At this moment, the web version have more features than the spreadsheet version, for examples, sample size calculators for confidence interval of Pearson's correlation, Intraclass correlation coefficient and Cronbach's alpha coefficient. It also includes the sample size for animal study.
